New Server Folders

How Does it work?

Here are some steps to create server forlder.

To create a server folder, click, hold and drag a server over another server for a few seconds and drop it in.

You can access your Folder Settings as well as Mark Server Folder as Read by right clicking on the server folder.

In the Folder Setting window, you can rename the folder and change the folder color.

You can add additional servers into existing folders by dragging and dropping them into the folder.

A folder can hold up to 100 servers!

To remove a server from a folder, simply drag it away from the folder. Once a folder has no servers within, it will be deleted.
